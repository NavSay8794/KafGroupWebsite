# FreshPak Supply Landing

A Next.js App Router landing website scaffold for a fresh produce supplier and exporter.

## Project structure

- `app/layout.tsx` — root layout with global metadata, header, footer, and stylesheet.
- `app/page.tsx` — home landing page with hero, about, services, export, vendors, testimonials, and contact.
- `app/about/page.tsx` — stub page for About.
- `app/services/page.tsx` — stub page for Services.
- `app/export/page.tsx` — stub page for Export.
- `app/vendors/page.tsx` — stub page for Vendors.
- `app/contact/page.tsx` — stub page for Contact.
- `components/Header.tsx` — responsive navigation with mobile toggle.
- `components/Footer.tsx` — footer links and contact details.
- `components/ContactForm.tsx` — client form component with state.
- `components/ServiceCard.tsx` — reusable service card.
- `app/globals.css` — responsive layout, styling, and accessibility-friendly rules.

## Setup

```bash
npm install
npm run dev
```

## Notes

This App Router structure is ready to expand with new pages and route-specific content.
